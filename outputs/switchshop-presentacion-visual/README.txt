Presentacion visual SwitchShop

Archivo principal:
presentacion-visual-switchshop.html

Como usarla:
1. Abre el archivo en Chrome o Edge.
2. Presenta desplazandote verticalmente por las secciones.
3. Para exportar como PDF: Ctrl+P > Guardar como PDF.
4. Para capturas por modulo puedes abrir el mismo archivo con estos parametros:
   ?shot=cover
   ?shot=value
   ?shot=store
   ?shot=dashboard
   ?shot=inventory
   ?shot=orders
   ?shot=customers
   ?shot=receivables
   ?shot=logistics
   ?shot=settings
   ?shot=users
   ?shot=modules
   ?shot=close

Nota:
En este equipo Edge/Chrome headless fallo por GPU, por eso no se pudieron exportar PNGs automaticos desde Codex. El HTML esta preparado para verse como slides y exportarse desde el navegador.
